## Open Items

### Lab-Grown Diamonds (~29,333 unlabeled)

The `unlabeled/` directory contains lab-grown diamond images with full 4C attributes in `diamonds_lab_raw.csv` but no value tier labels. These were kept separate because lab-grown diamonds occupy a fundamentally different price space than naturals — applying natural-diamond percentile boundaries would produce meaningless tiers.

**Future uses (in scope per proposal Section 2.3):**

- **Separate tier labeling** — compute lab-grown-specific percentile boundaries from `diamonds_lab_raw.csv` and assign independent tier labels. Enables the natural vs. lab-grown accuracy delta metric (Section 4.4).
- **Semi-supervised pretraining** — use all 57K images (labeled + unlabeled) for self-supervised feature learning (SimCLR, DINO) before fine-tuning on labeled naturals. May improve representations for the small-tier classes.
- **Cross-origin generalization** — train on naturals, evaluate on lab-grown (or vice versa). Tests whether models learn genuine visual quality cues vs. price-correlated artifacts. Publishable finding either direction.
