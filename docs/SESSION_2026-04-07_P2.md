## 4. Domain Adaptation Experiment

### Design
Checkpoint: EfficientNetV2-S JA natural cross-domain (baseline F1=0.0708).
Pool: `be_natural_test.csv` (16,043 images, never seen during training).
Stratified 80/20 fine-tune train/val split within each N; remaining images
form held-out eval set.

Configuration: LR=1e-5, frozen backbone except last two blocks + classifier
head, float32, patience=3.

**Leakage verification:** All conditions confirmed clean. ft_train ∩ ft_val = 0,
ft_train ∩ eval = 0, ft_val ∩ eval = 0. ft_train + ft_val + eval = 16,043 =
pool total for all N.

Note: initial verification script had a parsing bug (`sampled_ids.json` read as
list; actual format is `{diamond_id: value_tier}` dict). Claude Code caught and
corrected before reporting results.

### Results

| Run | Fine-tune N | Epochs | Eval Macro F1 | vs Baseline |
|---|---|---|---|---|
| Baseline (zero-shot) | 0 | — | 0.0708 | — |
| N=500 | 400 train / 100 val | 10 | ~0.261 | +0.190 (3.7×) |
| N=1000 | 800 / 200 | 10 | ~0.320 | +0.249 (4.5×) |
| N=2000 | 1600 / 400 | 10 | 0.3723 | +0.302 (5.3×) |
| N=2000 ext | 1600 / 400 | 20 | 0.4004 | +0.330 (5.7×) |

N=2000 ext: best at epoch 19, not converged at epoch 20. Estimated asymptote
~0.42–0.43. Further extension not pursued (diminishing returns). Disclosed as
limitation.

### Interpretation
Domain gap is bridgeable with minimal target-domain data. 500 samples produce
3.7× F1 improvement. This is a calibration problem, not a representational one.

## 5. Regression Experiment — Alternative Formulation

### Design
- Architecture: EfficientNetV2-S
- Head: Linear(1280→1) with .squeeze(1) replacing Linear(1280→4)
- Loss: HuberLoss(delta=0.5) in log space
- Target: log(price_usd)
- Subsets: ja_natural, be_natural within-site
- Config: LR=3e-4, backbone scale 0.1, max 30 epochs, patience=5

### Training Outcomes

| ja_natural | 0.3300 | 14 | 19 | Yes |

## 7. Brightness-Accuracy Correlation Analysis

Inference run on 2000 BE images using EfficientNetV2 JA natural cross-domain
checkpoint (~30 seconds GPU). No predictions CSV existed — run on the fly.

### Results

| Feature | Spearman ρ | p-value |
|---|---|---|
| r_mean | +0.110 | <0.0001 |
| g_mean | +0.088 | <0.0001 |
| b_mean | +0.090 | <0.0001 |
| r_std | −0.030 | 0.17 (n.s.) |

Model predicted "premium" for 93.7% of all 2000 BE inputs (0 budget, 19
investment_grade, 108 mid_range, 1873 premium).

Per-tier breakdown:
- budget (n=515): 0% accuracy
- investment_grade (n=203): 0.5% accuracy (1 correct image)
- mid_range (n=992): 5.7% accuracy, ρ=+0.039 (n.s.)
- premium (n=290): 94.1% accuracy — only because model predicts premium for everything

### Interpretation
Brightness-accuracy correlation (ρ≈0.09–0.11) reflects tier composition, not
per-image discriminability. Brighter BE images are disproportionately premium-tier;
premium is the only class the collapsed model predicts.

**Key conclusion:** Cross-domain failure is categorical, not continuous. The
brightness shift (Cohen's d=0.77) displaces BE inputs outside the model's learned
decision space entirely, causing collapse to near-constant output. This directly
motivates the adaptation experiment and should be framed as such in the paper.
